SmartSpend AI 💰

A full-stack personal finance platform built with Flutter and Firebase that helps users track expenses, manage investments, monitor savings goals, and gain actionable financial insights through a custom Local Intelligence Engine.

🚀 Overview

SmartSpend AI is designed to simplify personal finance management by combining expense tracking, investment portfolio management, goal-based savings, financial analytics, and intelligent recommendations into a single mobile-first experience.

Unlike generic finance apps that only record transactions, SmartSpend AI analyzes user financial behavior and generates personalized insights based on real financial data.
